# student-performance-predictor-ml
🎓 Machine Learning project that predicts student pass/fail using study hours, attendance, and previous scores with a Flask web app interface.


---

## 🔥 Live Features
✔ Predict pass/fail instantly  
✔ Clean and simple UI  
✔ Trained ML model (Logistic Regression)  


## 🚀 Overview
This project predicts whether a student will pass or fail based on:
- Hours studied
- Attendance
- Previous scores

## 🛠 Tech Stack
- Python
- pandas
- scikit-learn
- Flask (for web app)

## 📊 Features
- Data cleaning & preprocessing
- Logistic Regression model
- Accuracy evaluation
- Web-based prediction UI

## 🧠 Model
Used Logistic Regression to classify pass/fail.
